# Alke Wallet

Proyecto Módulo #2 | ABP

## Descripción
Alke Wallet permite a los usuarios:
- Crear y gestionar cuentas.
- Realizar transacciones seguras.
- Visualizar saldo y movimientos.
- Navegar por una interfaz sencilla y funcional.
